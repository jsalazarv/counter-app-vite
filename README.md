### Built with

- ✨ React ✨
- ✨ Tailwind ✨
- Vite

### Install

Clone the project with the following command:

```bash
git clone https://github.com/jsalazarv/counter-app-vite.git
```

Install the dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
